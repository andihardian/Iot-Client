Aplikasi IoT Pintu Pintar Berbasis Pengenalan Wajah adalah sistem keamanan modern yang memungkinkan pintu terbuka secara otomatis menggunakan teknologi face recognition berbasis Raspberry Pi. Sistem ini memanfaatkan kamera dan kecerdasan buatan untuk mengenali wajah pengguna yang telah terdaftar, sehingga akses menjadi lebih aman, cepat, dan praktis tanpa perlu menggunakan kunci fisik.

Aplikasi ini menggunakan Raspberry Pi sebagai pusat pengendali utama yang terhubung dengan kamera, sensor pintu, relay, dan solenoid door lock. Seluruh sistem dapat dipantau secara real-time melalui smartphone atau web dashboard. Pengguna dapat menambahkan data wajah, melihat riwayat akses, menerima notifikasi ketika ada percobaan akses tidak dikenal, serta mengontrol pintu dari jarak jauh melalui jaringan internet.
